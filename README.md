# Анализ лояльности пользователей Яндекс Афиши с помощью Python

## Описание проекта
Основная цель проекта — провести исследовательский анализ данных для отдела маркетинга, чтобы выявить закономерности в поведении пользователей. Исследование помогает понять, какие клиенты с большей вероятностью возвращаются на платформу и совершают повторные покупки.

**Результаты проекта позволяют:**
- Быстро выявлять перспективных клиентов и предлагать им персонализированные условия.
- Точно настраивать рекламу на аудитории с высокой вероятностью возврата.
- Оптимизировать маркетинговые бюджеты.
- Повысить общий уровень удержания клиентов (Retention Rate).

## Технологический стек
- **Язык:** Python
- **Библиотеки:** pandas, numpy, matplotlib, seaborn, sqlalchemy, phik
- **База данных:** PostgreSQL
- **Безопасность:** python-dotenv (скрытие учетных данных базы данных)
