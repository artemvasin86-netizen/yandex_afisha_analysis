# Анализ лояльности пользователей сервиса «Яндекс Афиша»

 Описание проекта
Проект посвящен глубокому исследовательскому анализу данных (EDA) для отдела маркетинга «Яндекс Афиши». Основная цель - выявить закономерности в поведении пользователей, определить портрет лояльного клиента и разработать рекомендации по повышению удержания (Retention Rate).

Исследование помогает бизнесу:
- Сегментировать базу: выделять перспективных клиентов для персональных предложений.
- Оптимизировать бюджет: точнее настраивать рекламные кампании на аудитории с высоким потенциалом возврата.
- Увеличить LTV: внедрять триггерные коммуникации в критические моменты жизненного цикла клиента.

 Ключевые результаты исследования
- Качественная предобработка: Проведена очистка данных от аномалий и «тяжелых хвостов» (фильтрация по 99-му перцентилю). Это позволило снизить шум в данных и уменьшить стандартное отклонение в 1.7 раза, сделав выводы более точными.
- Социальный паттерн: Выявлено, что ядром лояльной аудитории являются пользователи, покупающие 2-3 билета за раз. Это указывает на семейный или групповой характер посещения мероприятий - важный инсайт для таргетинга.
- Корреляционный анализ: С помощью коэффициента $\phi_k$ (Phik) подтверждена значимая связь между сегментом количества заказов и частотой возврата на платформу.
- Бизнес-рекомендации: Сформулированы стратегии удержания, включая запуск триггерных рассылок на 7-10 день после покупки и технические улучшения (внедрение биометрии и автозаполнения) для снижения «трения» при оплате.

 Технологический стек
- Язык: Python
- Библиотеки: 
  - pandas, numpy - обработка и анализ данных.
  - matplotlib, seaborn - визуализация данных.
  - sqlalchemy - работа с базой данных PostgreSQL.
  - phik - расчет продвинутых коэффициентов корреляции.
  - python-dotenv - безопасность и работа с переменными окружения.
- База данных: PostgreSQL

 Инструкция по запуску
Для запуска проекта локально выполните следующие шаги:

1. Клонируйте репозиторий:
   git clone https://github.com/artemvasin86-netizen/yandex_afisha_analysis.git
   cd yandex_afisha_analysis
2. Настройте окружение:
  Создайте виртуальное окружение и установите зависимости:
   python -m venv Work_Env
   
   # Активация для Windows:
   Work_Env\Scripts\activate
   
   # Активация для macOS/Linux:
   source Work_Env/bin/activate

   pip install -r requirements.txt
3. Настройте безопасность:
  Создайте в корневой папке файл .env (он добавлен в .gitignore и не попадет в сеть) и добавьте в него свои доступы к БД:
   DB_HOST=ваш_хост
   DB_USER=ваш_логин
   DB_PASSWORD=ваш_пароль
   DB_NAME=имя_базы
   DB_PORT=порт
4. Запустите анализ:
  Откройте DataAnalysis.ipynb через Jupyter Notebook или VS Code.

 Структура репозитория
- DataAnalysis.ipynb - основной файл с кодом, графиками и выводами.
- requirements.txt - список библиотек, необходимых для работы проекта.
- .gitignore - список исключенных файлов (включая конфиденциальные данные .env и папку drafts).
- README.md - документация проекта.
